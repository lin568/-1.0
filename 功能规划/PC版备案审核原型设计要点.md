#### 设计要点

**筛选区域**
- 下拉选择器使用Element Plus Select组件
- 时间范围使用DatePicker,支持快捷选项(今天/本周/本月/自定义)
- 搜索框支持工程名称、工程地址、申报人模糊搜索
- 筛选条件实时生效,无需点击"查询"按钮
- 已选筛选条件以Tag形式展示,可快速清除

**数据统计卡片**
- 4个关键指标卡片,使用不同颜色区分
- 待审核(橙色 #fa8c16)、今日新增(蓝色 #1890ff)、本周审核(绿色 #52c41a)、通过率(紫色 #722ed1)
- 数字大号显示(32px),标签小号显示(14px)
- 卡片hover时有阴影效果

**数据表格**
- 表头固定,内容区域可滚动
- 行高48px,保证舒适的点击区域
- 奇偶行背景色区分(#ffffff / #fafafa)
- hover行高亮显示(#e6f7ff)
- 审核状态使用Tag组件:待审核(橙色)、已通过(绿色)、已驳回(红色)
- 操作列固定在右侧,按钮使用primary类型

**分页器**
- 使用Element Plus Pagination组件
- 支持每页显示50/100/200条切换
- 支持跳转到指定页
- 显示总条数和当前范围

**页面头部**
- 返回按钮使用图标+文字,点击返回列表页
- 工程名称居中显示,字号18px,加粗
- 审核历史下拉菜单,显示历次审核记录(审核人、时间、结果、原因)

**左侧信息展示区**
- 使用Collapse折叠面板组件,默认全部展开
- 每个信息块包含:标题、字段列表、驳回按钮
- 字段采用"标签:值"的形式展示,标签宽度120px,值自适应
- 重要字段高亮显示(如工程总额、建筑面积)
- OCR识别字段显示对比图标,hover显示识别结果与填写值的对比
- 驳回按钮为次要按钮,点击后弹出驳回原因输入框
- 已驳回字段标红显示,显示驳回原因

**右侧材料预览区**
- 上方材料列表,使用Tree组件展示材料分类
- 已上传材料显示绿色勾选图标
- 未上传材料显示灰色图标
- 点击材料名称,下方预览区显示对应材料
- 预览区支持图片/PDF预览
- 图片预览:支持放大、缩小、旋转、全屏
- PDF预览:使用pdf.js,支持翻页、缩放、下载
- 预览区底部显示操作按钮:放大、缩小、旋转、下载、全屏

**AI审核意见区**
- 使用Alert组件展示,类型为warning
- 显示AI识别的风险点和建议
- 可折叠,默认展开

**底部操作栏**
- 固定在页面底部,背景色#ffffff,上边框阴影
- 驳回按钮:danger类型,点击后弹出驳回确认弹窗
- 通过按钮:primary类型,点击后弹出通过确认弹窗
- 保存草稿按钮:default类型,保存当前审核进度

## 三、交互设计规范

### 3.1 按钮交互

**按钮类型**
- Primary按钮:主要操作(通过、确认、提交)
- Danger按钮:危险操作(驳回、删除)
- Default按钮:次要操作(取消、保存草稿)

**按钮状态**
- Normal:默认状态
- Hover:鼠标悬停,背景色加深10%
- Active:点击时,背景色加深20%
- Disabled:禁用状态,灰色,不可点击
- Loading:加载中,显示loading图标,禁止点击

**按钮尺寸**
- Large:高度40px,用于重要操作
- Default:高度32px,用于常规操作
- Small:高度24px,用于表格内操作

### 3.2 表单交互

**输入框**
- Focus时显示蓝色边框
- Error时显示红色边框和错误提示
- 必填项标签前显示红色星号

**下拉选择器**
- 支持搜索过滤
- 选中项显示勾选图标
- 支持清空选择

**日期选择器**
- 支持快捷选项(今天/本周/本月)
- 支持自定义范围选择
- 显示格式:YYYY-MM-DD

### 3.3 反馈交互

**消息提示**
- Success:绿色,操作成功提示
- Warning:橙色,警告提示
- Error:红色,错误提示
- Info:蓝色,信息提示
- 显示位置:页面顶部居中
- 自动关闭时间:3秒

**加载状态**
- 全局加载:页面中央显示loading图标
- 局部加载:组件内显示loading图标
- 按钮加载:按钮内显示loading图标

**确认弹窗**
- 重要操作前弹出确认
- 显示操作影响说明
- 提供取消和确认按钮

## 七、开发注意事项

### 7.1 技术栈

- 前端框架: Vue 3
- UI组件库: Element Plus
- 状态管理: Pinia
- 路由: Vue Router
- HTTP请求: Axios
- PDF预览: pdf.js
- 图表: ECharts (如需要)

### 7.2 浏览器兼容

- Chrome ≥90
- Edge ≥90
- Firefox ≥88
- Safari ≥14

### 7.3 权限控制

- 审核人员只能查看所属街镇的备案
- 操作日志记录所有审核操作

- 所有请求使用HTTPS
- 敏感数据加密传输
- 文件上传大小限制10MB
