## 📱 交互说明

### 页面跳转流程
```
选择入驻方式
  ├─ 法人入驻 → 填写企业信息 → 确认提交 → 提交成功弹窗 → 申请记录列表
  └─ 授权入驻 → 填写企业信息 → 填写申请人信息 → 提交成功弹窗 → 申请记录列表

申请记录列表
  ├─ 审核中 → 申请详情(审核中) → 可撤回
  ├─ 已通过 → 申请详情(已通过) → 查看企业详情
  └─ 已驳回 → 申请详情(已驳回) → 重新提交

我的企业列表
  └─ 点击企业 → 企业详情
                  ├─ 补充资料 → 补充资料页面
                  ├─ 成员管理 → 成员管理页面
                  └─ 查看备案 → 备案列表页面
```

### 弹窗触发时机
1. **提交成功弹窗**: 点击"提交申请"按钮后
2. **审核通过通知**: 后台审核通过后推送
3. **审核驳回通知**: 后台审核驳回后推送
4. **邀请成员弹窗**: 点击"邀请成员"按钮
5. **移除成员确认**: 点击"移除"按钮
6. **撤回申请确认**: 点击"撤回申请"按钮
7. **数据校验提示**: 选择施工单位时数据不完整
8. **禁用企业提示**: 选择已禁用的施工单位
